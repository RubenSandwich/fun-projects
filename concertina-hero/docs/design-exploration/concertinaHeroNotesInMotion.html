3 · keys stay visibleEvery incoming key lights up

The moment a note appears at the top, its target button lights with the note's colour and letters — so you can find the key long before the note lands, not just as it arrives.

Reading it: each horizontal band is one beat; a note's height is how long to hold. A tall bar feeds through the hit line as you hold it — the button glows and a spark sits on the line while points build for the fraction you actually sustain.
